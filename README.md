# talk-to-text-summary-by-openai-api 
OpenAI Whisperを使ってmp3データを文字起こし，ChatGPTで整形，要約するプログラムです．

## requirements
- openai
- subprocess
- pydub
- ffmpeg
  - mac os
```
brew install ffmpeg
```
